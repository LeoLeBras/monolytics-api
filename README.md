# monolytics-api
The official Monolytics API :tv:

## Requirements
- PHP 5.4.x or newer

## Fetch data from :
- Youtube
- Twitter
- Trakt
- OMDb
- Rotten Tomatoes
- PirateBay

## Make the awesome

#### Install
Just [clone](github-windows://openRepo/https://github.com/LeoLeBras/monolytics-api.git) the repo
and start :
```shell
$ git clone https://github.com/LeoLeBras/monolytics-api.git
```

#### Run
Open ```index.php``` file.
